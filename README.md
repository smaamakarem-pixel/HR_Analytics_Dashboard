# HR Analytics Dashboard

An end-to-end HR analytics project analyzing workforce data for **161 employees**, built using three different tools — **Power BI**, **Excel**, and **Tableau** — to demonstrate flexibility across BI platforms.

## 📊 Overview

The dashboard provides insights into employee compensation, demographics, performance, and leave management to support HR decision-making.

## 🔍 Key Insights

- **Salary by Education**: Employees with a Master's degree earn the highest average salary, followed by Bachelor's, Diploma, and High School holders.
- **Top-Paying Roles**: Product Manager and Research Scientist roles command the highest salaries (up to $85,000).
- **Workforce Demographics**: Gender split of 54.66% female and 45.34% male, with the majority of employees aged between 25–40.
- **Headcount Trend**: Employee count grew steadily from 2018, peaking around 2020, then declined slightly in subsequent years.
- **Leave Balance Analysis**: Leave balances vary by job title, with Packaging Associates showing the highest combined leave balance among roles.

## 🛠️ Tools & Skills Used

- **Power BI**: Data modeling, DAX measures, interactive slicers and filters
- **Excel**: Pivot Tables, Power Query, Power Pivot, data visualization
- **Tableau**: Calculated fields, dashboard design, interactive filtering
